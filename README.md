## 🎡 Ferris Wheel CSS Animation Project

### 📌 Preview  
This project is a CSS animation exercise where I built a Ferris wheel to practice creating smooth motion, transforming elements, and controlling animation timing using only CSS.

---

### 📖 About the Project  
This project focuses on learning how CSS animations can bring static webpages to life. The goal was to construct a Ferris wheel and animate its movement using CSS properties such as transforms and keyframes.

The exercise demonstrates how animation can be used to draw attention to specific UI elements and make a webpage more interactive and visually engaging—all without JavaScript.

---

### 🚀 What I Learned  
- How to create **CSS animations using keyframes**  
- Using **transform properties** to rotate and position elements  
- Controlling animation speed and timing with CSS  
- Structuring nested elements for complex shapes  
- Applying animation to enhance UI engagement  
- Understanding how pure CSS can simulate real-world motion  

---

### 🛠️ Technologies Used  
- HTML5  
- CSS3  
- CSS Animations (Keyframes)  
- CSS Transforms  

---

### 🎯 Key Takeaway  
This project helped me understand how powerful CSS alone can be for creating motion and interactivity, and how animation can significantly improve the visual appeal of a webpage.
